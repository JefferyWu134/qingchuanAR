# 晴川·映江梦 - 完整前端展示站点

> 武汉晴川阁沉浸式诗意 AR 体验项目 - 品牌展示网站

## 📖 项目简介

《晴川·映江梦》是一个融合 AR 技术与千年文化的创新文旅项目。本站点为完整的品牌展示网站，全面呈现项目的概念设计、视觉风格、功能模块与商业价值。

**设计理念：** 东方诗意 + 现代科技 + 高级美学

## 🗂️ 项目结构

```
晴川·映江梦/
├── index.html              # 首页门户
├── ar-experience.html      # AR 体验流程页
├── visual-style.html       # 视觉风格体系页
├── landmarks.html          # 多地标场景页
├── characters.html         # 角色与 IP 形象页
├── storyboard.html         # 剧情分镜设计页
├── assets.html             # 延展物料设计页
├── css/
│   ├── global.css          # 全局样式与主题变量
│   └── components.css      # 组件样式（导航、页脚等）
├── js/
│   ├── main.js            # 主要交互逻辑
│   └── animations.js       # 动画效果
├── assets/                 # 素材文件夹（预留）
└── README.md              # 项目说明文档
```

## 🚀 使用说明

### 1. 本地运行
直接用浏览器打开 `index.html` 即可查看完整站点。

```bash
# 如果需要本地服务器（可选）
# 使用 Python 3
python3 -m http.server 8000

# 或使用 Node.js 的 http-server
npx http-server -p 8000

# 然后访问 http://localhost:8000
```

### 2. 浏览建议
- **推荐浏览器：** Chrome、Firefox、Safari（最新版本）
- **最佳分辨率：** 1920×1080 及以上
- **移动端：** 完全响应式设计，支持手机和平板浏览

### 3. 导航使用
- 顶部导航栏可快速切换页面
- 移动端点击右上角按钮展开菜单
- 页面滚动时自动显示"返回顶部"按钮

## 📝 开发说明

### CSS 变量系统
全局样式使用 CSS 变量管理主题，便于统一修改：

```css
:root {
    --primary-blue: #1A3A5A;
    --primary-green: #1E5948;
    --accent-red: #8C3B32;
    --gold: #D4AF37;
    /* ... 更多变量 */
}
```

### 组件化思维
虽然使用原生 HTML，但代码组织遵循组件化思维：
- 导航栏 `.navbar`
- 卡片 `.card`
- 按钮 `.btn`
- 徽章 `.badge`
- 等等

### 动画控制
所有动画使用 CSS 实现，JavaScript 仅负责触发类名：
- `.scroll-reveal` - 滚动显示动画
- `.fade-in` - 淡入效果
- `.slide-up` - 上滑效果

## 🔧 自定义修改

### 修改配色
编辑 `css/global.css` 中的 CSS 变量即可全局更换配色。

### 修改内容
直接编辑各 HTML 文件中的文字内容，结构清晰，易于修改。

### 添加页面
1. 复制现有页面作为模板
2. 修改内容
3. 在所有页面的导航栏中添加链接

### 修改动画
编辑 `js/animations.js`，可以调整动画参数、添加新效果。
